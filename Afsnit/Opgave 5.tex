\section{Question 5}

\subsection{Stochastic spouse presence ($p_s=0.8$) and fertility only when spouse is present.}

\bigskip

\paragraph{Spouse presence.}

We extend the baseline model with a stochastic, income-generating spouse. When present, the spouse contributes income $y_t = 0.1 + 0.01t$ (as in Question~2); when absent, the household receives no spouse income. Presence is captured by a new state $s_t \in \{0,1\}$,

\[
s_t =
\begin{cases}
1 & \text{if a spouse is present,} \\
0 & \text{otherwise,}
\end{cases}
\]

which follows an i.i.d.\ process, written in the same form as the fertility transition,

\[
s_{t+1} =
\begin{cases}
1 & \text{with probability } p_s = 0.8, \\
0 & \text{with probability } 1-p_s = 0.2.
\end{cases}
\]

The household's state vector therefore expands to

\[
x_t = (s_t,n_t,a_t,k_t),
\]

with $s_t$ spouse presence, $n_t$ children, $a_t$ assets, and $k_t$ human capital.

\paragraph{Bellman equation and budget constraint.}

The household's dynamic optimization problem now incorporates stochastic spouse income. The Bellman equation becomes

\[
V_t(s_t,n_t,a_t,k_t)
=
\max_{c_t,h_t}
\left\{
\frac{c_t^{1+\eta}}{1+\eta}
-
\beta(n_t)\frac{h_t^{1+\gamma}}{1+\gamma}
+
\rho \, \mathbb{E}_t
\left[
V_{t+1}(s_{t+1},n_{t+1},a_{t+1},k_{t+1})
\right]
\right\}.
\]

The budget constraint is also modified because the household receives spouse income when $s_t=1$. 
The period budget constraint becomes

\[
a_{t+1} = (1+r)\left(a_t + (1-\tau_t)(w_t h_t+s_ty_t)-c_t\right).
\]

Thus, spouse income increases household resources only in periods when a spouse is present.

\paragraph{Fertility restriction.}

In addition to stochastic spouse presence, fertility is restricted so that births can only occur when a spouse is present. As in the baseline model, at most one birth can occur. Let $p_n$ denote the birth probability when the household is eligible.

If $s_t=1$ and $n_t=0$, a birth occurs with probability $p_n$:
\[
\Pr(n_{t+1}=n_t+1 \mid s_t=1,n_t=0)=p_n,
\]
\[
\Pr(n_{t+1}=n_t \mid s_t=1,n_t=0)=1-p_n.
\]

If a child is already present ($n_t\ge1$), fertility remains unchanged. When $s_t=0$, births cannot occur:
\[
\Pr(n_{t+1}=n_t \mid s_t=0)=1.
\]

Thus, stochastic spouse presence reduces the number of periods in which childbirth is feasible compared to the baseline model.

\paragraph{Full recursive formulation.}

Collecting the pieces, the household with state $x_t=(s_t,n_t,a_t,k_t)$ and controls $(c_t,h_t)$ solves

\[
V_t(s_t,n_t,a_t,k_t)
=
\max_{c_t,h_t}
\left\{
\frac{c_t^{1+\eta}}{1+\eta}
-
\beta(n_t)\frac{h_t^{1+\gamma}}{1+\gamma}
+
\rho \, \mathbb{E}_t\!\left[V_{t+1}(s_{t+1},n_{t+1},a_{t+1},k_{t+1})\right]
\right\},
\]

subject to the transitions

\[
a_{t+1} = (1+r)\left(a_t + (1-\tau)\left(w_t h_t + s_t y_t\right) - c_t\right),
\qquad
k_{t+1} = k_t + h_t,
\qquad
w_t = w(1+\alpha k_t),
\]

with $s_{t+1}$ and $n_{t+1}$ following the spouse and fertility processes above. Since spouse presence and the birth shock are independent, the expectation factorises. Writing $\mathbb{E}^{s}V_{t+1}(\,\cdot\,,n,a,k)=p_s V_{t+1}(1,n,a,k)+(1-p_s)V_{t+1}(0,n,a,k)$ for the expectation over next-period spouse presence, the continuation value is

\[
\mathbb{E}_t\!\left[V_{t+1}\right]=
\begin{cases}
p_n\,\mathbb{E}^{s}V_{t+1}(\,\cdot\,,n_t+1,a_{t+1},k_{t+1}) + (1-p_n)\,\mathbb{E}^{s}V_{t+1}(\,\cdot\,,n_t,a_{t+1},k_{t+1}) & \text{if } s_t=1,\ n_t=0, \\[4pt]
\mathbb{E}^{s}V_{t+1}(\,\cdot\,,n_t,a_{t+1},k_{t+1}) & \text{otherwise.}
\end{cases}
\]

The first case is the only one in which a birth can occur (spouse present and no child yet); otherwise next-period children equal current children. In the last period there is no continuation value, so the problem reduces to the static choice

\[
V_T(s_T,n_T,a_T,k_T)=\max_{h_T}\frac{c_T^{1+\eta}}{1+\eta}-\beta(n_T)\frac{h_T^{1+\gamma}}{1+\gamma}
\quad\text{s.t.}\quad
c_T = a_T + (1-\tau)\left(w_T h_T + s_T y_T\right).
\]

Setting $p_s=1$, $y_t=0$ and $\theta=0$ recovers the baseline model. This is exactly the program implemented in \texttt{DynLaborFertModel\_Aflevering.py}, where $s_t$ is carried as an extra dimension of the solution arrays and the continuation value is computed by averaging over the two next-period spouse states.

\paragraph{Simulation results and economic interpretation.}

Figure \ref{fig:stoch_spouse_profiles} compares life-cycle profiles in the stochastic spouse model ($p_s=0.8$, $y_t=0.1+0.01t$) against the pure baseline ($p_s=1$, $y_t=0$). Two forces shape the comparison. When the spouse is present, the extra income relaxes the budget constraint, which through an income effect raises consumption and pulls down own hours. Because that income is now uncertain, households also face income risk and save somewhat more for precaution, which works in the opposite direction on hours. The net change in labour supply is therefore the balance of these two channels and is read off the figure; consumption is unambiguously higher whenever spouse income is realised. Fertility falls relative to the baseline, since births are only possible in the fraction of periods in which a spouse is present.
